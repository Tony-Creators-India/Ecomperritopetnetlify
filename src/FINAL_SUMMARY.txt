   ❌ Thank You Page Shows 404
   ─────────────────────────────────────────────────────────────────────────
   Problem: File not deployed
   
   Solution:
   1. Check: ls dist/thank-you.html
   2. If missing, rebuild: npm run build
   3. Verify vite.config.ts exists
   4. Redeploy


   ❌ Payment Doesn't Redirect
   ─────────────────────────────────────────────────────────────────────────
   Problem: Wrong redirect in code
   
   Solution:
   1. Check /components/RazorpayPaymentIframe.tsx line 45
   2. Should be: window.location.href = '/thank-you.html';
   3. Rebuild and redeploy


   ❌ Back Button Doesn't Work
   ─────────────────────────────────────────────────────────────────────────
   Problem: Wrong URL in button
   
   Solution:
   1. Open /thank-you.html
   2. Find: <a href="..." class="cta-button">
   3. Update href to your domain
   4. Redeploy (no rebuild needed)
