                    BUILD PROCESS

═══════════════════════════════════════════════════════════════


When you deploy, Vercel will:

  1. npm install          → Install dependencies
  2. npm run build        → Build your app
     ├── prebuild         → Auto-verify _redirects
     └── vite build       → Create dist/ folder
  3. Deploy dist/         → Upload static files
  4. Provide live URL     → Your site is live! ✅


Expected output:

  dist/
  ├── index.html              ✅
  ├── _redirects              ✅
  └── assets/
      ├── index-[hash].js     ✅
      └── index-[hash].css    ✅
