                    TROUBLESHOOTING

═══════════════════════════════════════════════════════════════


Build fails?
  rm -rf node_modules package-lock.json
  npm install
  npm run build

No dist folder?
  Make sure 'npm run build' completed without errors.

Routes don't work?
  Make sure you uploaded the entire 'dist' folder,
  not just the files inside it.
